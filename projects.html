LockBox

A user-friendly desktop application designed to enhance the protection of sensitive data, making digital security accessible to all.

It contains 10 tools, all geared towards various data security protection

Created using:
  • Python
  • PyQt5
  • SQLite
I worked on 3 tools:
  • Metadata Deletion: Erases hidden metadata from files to prevent unauthorized use, distribution, modification.
  • Secure Notes: Automatically erases notes after a set time to prevent unauthorized access and ensure information privacy.
  • Secure Messaging: A secure two-person chat app where messages vanish once either user closes the app, with both entering a shared code to begin a private conversation.

EarthEcho

A blogging platform where users can share tips and insights on sustainable living, fostering a community of like-minded individuals.

Users are able to create accounts, login and interact with the community to earn points and get discounts on sustainable shopping.

Created using:
  • Python
  • Flask
  • MySQL
  • HTML, CSS, JS, Bootstrap
I worked on:
  • Authentication & Authorization: User and admin signup, login, admin user management
  • Minigame: A small game made in godot for users to earn points
  • Points Shop & Vouchers: Incentives for users to stay on the platform to earn discounts
  • Others: Messaging, Favourites
